# Atividade de Progamação Orientada a Objetos

**Objetivo**
Realizar a atividade referente as páginas 12 e 13 do slide POO-5-Relacionamentos.pdf, e o complemento "Carrinho de compras", no Google Sala de Aula.

É pedido para criar quatro classes:

1. Produto: Contém os atributos id, valor, nome e descrição.
2. ItemPedido: Contém os atributos id, produto e quantidade.
3. Carrinho: Contém o atributo item.
4. Pedido: Contém os atributos carrinho e valor_total.

Foram feitos métodos de adicionar e remover qunatidade de produto, buscar por produto, remover produto, calcular valor total e outras auxiliares.

A ideia é, no futuro, implementar uma interface web pelo framework [Angular](https://angular.dev/).
